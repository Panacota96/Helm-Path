# The Forge 🔨🛡️

This directory contains the blueprints for **The Helm**, the isolated environment where the Watcher maintains his vigil.

## 🛡️ The Helm (Dockerfile.kali)
The **Helm** is a custom-forged Kali Linux environment designed for maximum vigilance and protection.

### **Features:**
- **The Watcher's Arsenal:** A comprehensive collection of 166+ security tools, including `nmap`, `metasploit`, `sqlmap`, `ffuf`, and specialized utilities like `linpeas` and `pspy`.
- **Interactive Sentry:** Pre-configured with `zsh` and `oh-my-zsh` for advanced tab-completion, themes, and a superior interactive experience.
- **The Watcher's Path (Logging):** A transparent, shell-level logger that records every command and its output to a persistent volume shared with the host machine.
- **Vigilance Hook:** The logging mechanism is integrated into the shell's initialization (`.zshrc`), ensuring no step of the vigil is ever forgotten.

## 🔨 Forging the Helm
The Helm is automatically built by the `helm-path start` command if it does not already exist. To manually trigger a rebuild of the Forge, you can use:

```bash
docker build -t helm-path:kali -f docker/Dockerfile.kali .
```

## 📜 Accessibility & Standards
- **High-Contrast Interface:** The terminal is configured for readability, with clear, color-coded prompts and indicators.
- **Text-First Design:** The logging mechanism is text-based and captures all terminal output, ensuring the final chronicles generated by the Scribe are fully accessible.
- **Robust Tools:** By using standard, well-documented security tools, we ensure a predictable and accessible workflow for all users.
